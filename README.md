# jepzsh

*made with love by jep*

### Usage

simply run the `copy.sh` to move all the files in your home dir

### File list

- `.kbd.zsh` used for enforcing the correct kb map
- `.p10k.zsh` my `p10k` theme
- `.zlogin` used for managing the `$PATH`
- `.zshrc` main zsh file

### Dependencies

- eza
- fzf
- git
- zsh
